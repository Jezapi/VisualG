Algoritmo "QuadradoDaSoma"
   // Declaração de variáveis
   Declare A, B, C, Soma, Resultado: Real
   // Entrada de dados
   Escreva "Digite o primeiro valor (A):"
   Leia A
   Escreva "Digite o segundo valor (B):"
   Leia B
   Escreva "Digite o terceiro valor (C):"
   Leia C
   // Cálculo da soma
   Soma ? A + B + C
   // Cálculo do quadrado da soma
   Resultado ? Soma * Soma
   // Saída de dados
   Escreva "O quadrado da soma dos valores é: ", Resultado
FimAlgoritmo
